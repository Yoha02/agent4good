================================================================================
COMMANDS TO COMPLETE SETUP - Copy and paste these in Cloud SDK Shell
================================================================================

Step 1: Set up Application Default Credentials (CRITICAL!)
-----------------------------------------------------------
gcloud auth application-default login

(Your browser will open - log in and click "Allow")


Step 2: Set the project
-----------------------------------------------------------
gcloud config set project qwiklabs-gcp-00-86088b6278cb


Step 3: Set environment variable
-----------------------------------------------------------
$env:GOOGLE_CLOUD_PROJECT = "qwiklabs-gcp-00-86088b6278cb"


Step 4: Verify everything works
-----------------------------------------------------------
gcloud config get-value project
gcloud auth application-default print-access-token


Step 5: Navigate to project and run the agent
-----------------------------------------------------------
cd C:\Users\asggm\Agents4Good\agent4good
python interactive_demo.py


================================================================================
VERIFICATION CHECKLIST
================================================================================

After running the commands above, verify:

1. "gcloud config get-value project" shows: qwiklabs-gcp-00-86088b6278cb
2. "gcloud auth application-default print-access-token" returns a token (starts with "ya29...")
3. No error messages

If all checks pass, run: python interactive_demo.py

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Could not find default credentials"
Solution: Make sure you ran "gcloud auth application-default login"

Problem: Agent still fails with auth error
Solution: Make sure you're in the Cloud SDK Shell or have opened a NEW PowerShell window

Problem: "Module not found: google.adk"
Solution: Run "pip install google-adk google-genai"
